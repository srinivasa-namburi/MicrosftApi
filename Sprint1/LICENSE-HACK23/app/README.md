## Usage
1. Open appsettings.development.json
2. Enter completion keys, endpoints and model names (embedding values are not needed right now)
3. dotnet run the app and you should see console output highlighting the important parts of the hardcoded section

There are lots of comments in Program.cs on areas we can explore next.

!IMPORTANT!
Please do not commit any Azure OpenAI keys into the repository, or any other service keys for that matter.
